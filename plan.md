# QuickerUse - 项目功能规划文档

## 项目简介

**QuickerUse** 是一款基于 Electron + Vue 3 构建的桌面效率工具，采用"鼠标优先"设计理念，通过智能感知剪贴板内容、轮盘菜单快捷操作、自定义工具管理等功能，帮助用户快速完成日常开发和办公任务。

---

## 一、核心功能模块

### 1.1 智能推荐系统
- **剪贴板智能感知**：自动分析剪贴板内容，智能推荐相关操作
- **环境感知**：根据当前前台应用（如 VSCode、Chrome）自动调整推荐功能顺序
- **自定义排序**：支持拖拽调整功能显示顺序
- **功能隐藏/恢复**：可隐藏不常用功能，在设置中恢复

### 1.2 轮盘菜单（Radial Menu）
- **可视化配置**：8 扇区 × 3 层，最多支持 24 个快捷功能
- **数字键快捷**：底部 8 个数字键（1-8）快速触发系统功能
- **多种触发方式**：
  - 全局快捷键（默认 Alt+Space）
  - 鼠标中键点击
  - 应用内右键长按
- **主题支持**：深色/浅色/蓝色/紫色四种主题
- **动态半径**：支持 80-200px 轮盘大小调节

### 1.3 自定义工具区
- **文件/程序启动器**：拖拽 exe/lnk/bat 等文件添加快捷启动
- **内置功能集成**：可将常用内置功能添加到工具区
- **图标自动获取**：自动提取程序图标显示
- **管理员权限**：支持以管理员身份运行程序
- **快捷键绑定**：为自定义工具设置专属快捷键

## 二、内置功能清单

### 2.1 搜索与翻译
| 功能 | 描述 | 快捷操作 |
|------|------|----------|
| 搜索 | 使用默认浏览器搜索选中文本 | 支持 Google/百度/Bing/DuckDuckGo |

|------|------|
| JSON 处理 | 格式化/压缩/校验 JSON 数据 |
| YAML 处理 | YAML 格式化/校验/转 JSON |
| SQL IN | 多行文本转 SQL IN 语句格式 |

### 2.3 开发辅助
| 功能 | 描述 |
|------|------|
| 时间戳转换 | 时间戳与日期时间互转 |
| 变量命名 | 驼峰/下划线/短横线命名互转 |
| 正则助手 | 正则表达式测试和常用模式 |

### 2.4 信息处理
| 功能 | 描述 |
|------|------|
| 信息提取 | 从文本中提取 IP/邮箱/手机号/链接 |
| OCR 识别 | 识别图片中的文字内容（基于 Tesseract.js） |

| 二维码生成 | 将文本或链接生成二维码图片 |

### 2.6 系统工具
| 功能 | 描述 |

| 按键 | 功能 |
|------|------|

## 三、系统功能

### 3.1 窗口管理
- **托盘最小化**：点击关闭按钮最小化到系统托盘
- **窗口置顶**：支持窗口始终置顶显示
- **跟随鼠标**：唤醒时窗口跟随鼠标位置显示

### 3.2 快捷键系统
- **全局唤醒**：默认 Alt+Space，可自定义
- **功能快捷键**：为每个内置功能设置独立快捷键
- **自定义工具快捷键**：为添加的工具设置快捷键

### 3.3 启动与行为
- **开机自启**：支持开机自动启动
- **启动最小化**：启动后直接最小化到托盘
- **中键唤醒**：鼠标中键点击唤醒（仅 Windows）
- **召唤模式**：可选弹框模式或轮盘模式

### 3.4 数据管理
- **密钥管理**：安全存储 API Key 等敏感信息（PIN 码保护）
- **配置持久化**：所有设置自动保存到 localStorage

## 四、技术架构

```
QuickerUse/
├── electron/                 # Electron 主进程
│   ├── main/
│   │   ├── index.js          # 主进程入口
│   │   ├── systemTools.js    # 系统工具（PowerShell/robotjs）
│   │   ├── clipboardHistory.js # 剪贴板历史管理
│   │   ├── configManager.js  # 配置管理
│   │   └── secretManager.js  # 密钥管理
│   └── preload/              # 预加载脚本
├── src/                      # Vue 渲染进程
│   ├── components/
│   │   ├── GlobalRadialMenu.vue  # 全局轮盘菜单
│   │   ├── RadialMenuSettings.vue # 轮盘配置面板
│   │   ├── SettingsModal.vue     # 设置弹窗
│   │   ├── ToolGrid.vue          # 工具网格
│   │   ├── FeatureModal.vue      # 功能弹窗
│   │   └── ...
│   ├── utils/
│   │   ├── textProcessor.js  # 智能文本分析
│   │   ├── constants.js      # 功能常量定义
│   │   ├── envSensing.js     # 环境感知
│   │   └── ...
│   └── App.vue               # 主应用组件
└── resources/                # 资源文件
    └── MouseHook.exe         # 鼠标钩子（中键监听）
```
